您可以修改現有的磁碟區、包括變更磁碟區名稱、已分配的容量或服務層級。

步驟

  1. 登入 "NetApp Cloud Orchestrator"。

  2. 按一下您要管理的磁碟區名稱。

  3. 視情況修改下列Volume欄位：

    • 名稱

    • 標記

    • 已分配容量

    • 服務層級

      變更服務層級不會造成中斷、也不會影響用戶端資料存取。

請注意、可用的inode數量取決於分配的容量。

請參閱 "選取適當的服務層級和已分配的容量" 以取得詳細資料。
